Maneras de subir a Heroku una app

Usando heroku CLI.con los siguientes comandos:
  • heroku login
  • heroku git:clone -a luiscodoacodo
  • cd luiscodoacodo
  • git add .
  • git commit -am "nuestro comentario"
  • git push heroku master
  • Recuerda crear el archivo index.php e incluir de forma embebida el index.html
  • require_once("index.html")

  1. Pulsamos sobre connect to github
  2. Autorizamos a heroku
  3. Escribimos el nombre y pulsamos sobre connect
  4. Elegimos la rama y pulsamos sobre deploy branch
  5. Realiza de forma automática el deploy
