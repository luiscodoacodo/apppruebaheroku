Maneras de subir a Heroku una app

Usando heroku CLI.con los siguientes comandos:
  • heroku login
  • heroku git:clone -a luiscodoacodo
  • cd luiscodoacodo
  • git add .
  • git commit -am "nuestro comentario"
  • git push heroku master
  • Recuerda crear el archivo index.php e incluir de forma embebida el index.html
  • require_once("index.html")
